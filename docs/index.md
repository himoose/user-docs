# Welcome to Hi, Moose Documentation

Welcome to the official user documentation for **Hi, Moose** - an affordable, no-BS search optimization platform for SMBs (not another LLM tracker).

## What is Hi, Moose?

Hi, Moose helps you optimize your content for AI-powered search engines like ChatGPT, Google AI Overviews, Perplexity, Claude, and Gemini. As traditional search evolves into AI-generated answers, Hi, Moose helps improve the likelihood that your brand gets cited, mentioned, and recommended by AI systems.
